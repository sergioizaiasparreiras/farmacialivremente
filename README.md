# farmacialivremente
E-commerce robusto desenvolvido para a Farmácia Homeopática Livremente, transformando o atendimento via WhatsApp em uma experiência digital completa. A plataforma integra médicos e clientes, otimizando o fluxo de orçamentos e vendas. O projeto prioriza a eficiência operacional e a usabilidade, modernizando processos vitais da farmácia.
